1.はじめに



実際にwindows上で動く実行形式ファイル（拡張子|*.exe）を作ってみましょう。
といっても始めのうちは コンソールアプリケーション ですが・・・
プログラミングにはMicrosoft Visual C++ 2008 Express Editionという開発環境を使います。
C++言語とは人にもわかりやすい言語で、高級言語の一つです。

そもそもプログラムはコードを書いただけでは動きません。
PCがわかるようにまぁ機械語なるものに変換しなければいけません。
この変換作業 や動かすために必要なファイルをくっつけるといった面倒なことを開発環境はしてくれます。
なので使わない手はないのです。

2.導入方法


まずMicrosoft Visual C++ 2008 Express Editionのページへ行きます。
①.オンライン版インストール
下のほうのMicrosoft Visual C++ 2008 Express Editionの
Web インストール (ダウンロード）をクリックします。（下図の赤枠内）
ファイルをダウンロードして実行するとインストールが始まります。
 ②.オフライン版インストール
右のほうのMicrosoft Visual C++ 2008 Express Editionの
オフライン版インストールをクリックします。（下図の赤枠内）
オフライン版は少々面倒ですがネット環境が無い場合はこちらです。
