今までのプログラムは一回一回いちいち実行しなけれけばいけません。
面倒ですので5回ぐらい何回も繰り返せれば便利です。

・回数が決まっているループ
回数を決めて繰り返す場合FOR〜NEXT

FOR 変数＝初期化式 TO 最小値  STEP  増加量  

      ループさせる文

NEXT 変数


という文を使います。
STEP増加量は省略できます(その場合1になる)
前作った、足し算のプログラムを改良して5回繰り返せるようにしましょう

FOR I=1 TO 5
    INPUT A
    INPUT B
    LET C=A+B
    PRINT C
    PRINT I
NEXT I

END











Iがどのような値をとるのか見るために繰り返しの中にI自身の値を
表示させています。
Iは1→2→3→4→5と変化していきます。

これを利用すればいくつかまでの数の合計を求めるプログラムが作れます。

LET SUM=0

FOR I=1 TO 10
    SUM=SUM+I
NEXT I

PRINT SUM
END










これはSUMという変数に1〜10までの合計を入れて表示するプログラムです。

SUMにいままでのSUMとIの値を足していけば合計値になります。
TOの後(何回繰り返すか)のところには変数も使用できるので、
INPUT 文と組み合わせることでこちらからいくつまでの合計を
求めるか指定することもできます。


 

・回数が決まっていないループ
回数が決まっていないループはDO〜LOOPを使います。
ただし、これを使う場合は条件判断(IF文)を使ってEXIT DOを
実行しないといつまでもでられなくなって
終わらなくなってしまう(無限ループ)ので注意してください。

ちなみにFORループでもEXIT FORを実行すれば
回数に達していなくても終了できます。

LET SUM=0
LET I=0

DO
    LET I=I+1
    LET SUM=SUM+1
    IF I=10 THEN EXIT DO
LOOP

PRINT SUM
END













これは前のプログラムと同様の動きをします。
