・数当てゲームを作る
正解には乱数、入力にはINPUT、大きいか小さいか正解かをIFで判断して、
PRINTで表示、正解するまで繰り返す。
という今まで覚えたものをすべて使います。


REM * 数当てゲーム
RANDOMIZE
LET ANS=INT(RND*100+1)

DO
    INPUT PROMPT"予想される数字を入力(1〜100)":X
    IF X=ANS THEN EXIT DO
    IF X>ANS THEN
        PRINT "もっと小さい"
    ELSE 
        PRINT "もっと大きい"
    END IF
LOOP

PRINT "正解"
END


乱数の初期化をする。
ANSという変数に1〜100までの乱数を入れる。

INPUT PROMPTはPRINT文+INPUT文のような文で表示しながら入力を求めれます。
表示する文字と入力する変数は:（コロン）で区切ります。
XがANSと同じならばループからぬけ、正解と表示して終わります。

XがANSより大きければ「もっと小さい」と表示し、逆なら逆のことを表示します。


そしてもう一度入力してもらいます。



これでまず完成です。
