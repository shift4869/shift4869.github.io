・条件判断
・IF文
BASICには「もし〜ならば〜をする」という条件判断ができます。

この「もし〜ならば〜をする」というのを「IF〜THEN〜   END IF」と書きます

IF 条件式 THEN

  条件式が正しければ実行される文

END IF







IF文は条件式が正しければ実行される文が
1行のみならば、END IFは省略できます。

プログラムでどのように使えるかというと、
変数Aがあるとし、「もしAが10以上ならば"10以上"と表示する」と考えてみましょう。

LET A=10
IF A>=10 THEN
    PRINT "10以上"
END IF
END







>=は数学の≧と同じです。条件式にはこういった不等号や等号を使います。
下記のとおりです。

数学

＝

＞

＜

≧

≦

≠

BASIC

＝

＞

＜

＞＝

＜＝

＜＞または＞＜





上のプログラムの場合、Aは10で「Aが10以上」という条件に当てはまるので
THEN の後のPRINT"10以上"が実行されます。
もしAが10以下ならば「Aが10以上」という条件に当てはまらないので
THEN の後のPRINT"10以上"が実行されません。

LET A をINPUT Aにして自分で数値をいれてみてください。
10以下の場合何も表示されないはずです。


「もし〜ならば〜をする。でなければ〜をする」という文もあります。

IF〜THEN
    条件式が正しい場合の文
ELSE
    条件式が正しくない場合の文
END IF







この文です

さきほどのプログラムに「10以下ならば"10以下"と表示する」ように改良してみましょう。
IF A>=10 THEN PRINT "10以上"の次の行にIF A<10 THEN PRINT "10以下"と書いてもいいですが
IF〜THEN
ELSE
END IF
文を使ってみましょう。

INPUT A
IF A>=10 THEN 
    PRINT "10以上"
ELSE
    PRINT "10以下"
END IF
END









10以上でなければ10以下なのは当然です。
自分でいろいろ入力してみてください。


このIF文は今後もよく頻繁に使われます。
