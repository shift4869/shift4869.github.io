・組み込み関数
関数と聞くと難しそうですが、要はひとつの値を渡すと
違う値にして返すものだと考えればわかりやすいです。

簡単なものを4つ紹介します。
・INT( x )
xの小数点以下を切り捨てた整数を返します

・MOD( x , y )
xをyで割ったあまりを返します

・SQR( x )
xの平方根を返します

・ABS( x )
xの絶対値を返します

使用例を下に示します

REM * INT 使用例
LET x=3.141592763
LET y=0
LET y=INT( x )
PRINT "xの小数点以下を切り捨てるので";
PRINT y
END



REM * MOD 使用例
LET x=12
LET y=0
LET y=MOD( x , 5 )
PRINT "12(x)を5で割ったあまりは";
PRINT y
END　

　

　

REM * SQR 使用例
LET x=9
LET y=0
LET y=SQR(x)
PRINT "9(x)の平方根は";
PRINT y
END

 　

REM * ABS 使用例
LET x=-3
LET y=0
LET y=ABS(x)
PRINT "-3(x)の絶対値は";
PRINT y
END



REMはプログラムの実行に影響をあたえずに
メモを書くための文です。
ちなみに始め以外のところにメモを書きたい場合は、
!　(エクストラテンションマーク)の後に書けます。

PRINT文の後に　;　(セミコロン)をつけると
改行せずに続けて表示できます。


ほかにもいろいろな関数があります。
このようにはじめからある関数を
組み込み関数
といいます。
