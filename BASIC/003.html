前回のプログラム

PRINT "ハローBASIC"
END

のPRINT"文字"は画面上に文字を表示する文です。
プログラムは上から1行ずつ処理し、ENDに達したら終了します。



・変数
変数とはプログラムを書く上でなくてはならないものです。
変数はそれぞれ名前（識別子）がついた箱のようなものと考えるといいです。

次のプログラムをみてください

LET A=2
LET B=3
LET C=A+B
END

これは2+3＝？の足し算を想定したプログラムです。
A,B,Cという３つの変数を宣言しています。
変数の宣言はLET（識別子） ＝　(数値)　とします。

しかし、これを実行しても画面にはなにもでてきません。
画面に表示するPRINT文を書いていないからです。

「計算したんだからそれを見せるのが当然」と思うかもしてませんが
プログラムは命令されたことしか動きません、
逆に命令がしっかりしていれば忠実にやってくれます。
そういう意味でとても厳密です。

↑の計算結果をいれているCの値を表示するには、
LET C=A+Bの次の行に
PRINT C
と書けばいいですね。

変数はその名のとうり途中で値を変えることができます。

次のプログラムをみてください

LET A=2
LET A=3
PRINT A

LET B=2
LET B=B+3
PRINT B

END

さて、このプログラムでA,Bそれぞれの値はいくらになるでしょうか？
Aは始め2が入っていますが、次の文で3になりました。
なのでAは3です。2+3で5ではありません。

5という結果にしたいのならば、Bのように始め2が入っていて
次の文で（新しいB）＝（いままでのB）+3という文を実行します。
するとB＝2+3になるので5が表示されます。



・キーボードからの入力
前回の足し算をするプログラムではA＝2,B＝3としていました。
しかし、これでは違う足し算をしたい時にいちいちAとBの値を
書き換えなければいけません。
プログラムの実行時に指示できれば便利です。

そこでキーボードから入力する文INPUTを使います。
前のを改良してみましょう。

INPUT A
INPUT B
LET C=A+B
PRINT C
END

これを実行すると入力を促すように？と聞いてくるウィンドウがでてきます
そこにAの数値を入れエンターキーを押すと今度はBを聞いてきますので、同様に
してやると入力した2つの数値の和を表示してくれます。

四則演算の記号は+(プラス),-(マイナス)は同じですが
×(かける)÷(わる)はそれぞれ*(アスタリスク),/(スラッシュ)を使います。
Excelと同じですね。

どうだったでしょうか？もし実行できても結果がおかしいというようなことが
あったらステップ実行(実行ボタンの右隣かF10ボタン)で
１行ずつ変数の値を確認しながら実行し、直してください。
うまく動かないことをバグがあるといい、
修正することをデバックといいます。
ステップ実行はデバックにとても有効です。

最後に今まで覚えたLET,PRINT,INPUTは基本的な文です。
それゆえ、どんなプログラムでもこれらを使う機会が多くあるのしっかり覚えてください。
