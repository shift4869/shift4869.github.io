テトリミノの生成

テトリミノの生成はファクトリメソッドのようなものを創ります。
といっても、単なるTetriMinoクラスを正しく設定して返すっていうだけですけど・・
なかなか大きい関数になるので、これだけでソースファイル(*.cpp)を分割します。
ソリューションエクスプローラのsourceフォルダを右クリックして「追加->新しい項目」で、
CreateTetriMino.cppというファイルを創ってください。

それに書いていきます。
//CreateTetriMino.cpp
#include "../include/main.h"

//テトリミノ生成関数
TetriMino CreateTetriMino(const Point& p,const BLK_COL& k){
	TetriMino t;

	//tそのもののベース点
	t.x=p.x;
	t.y=p.y;

	//(2,2),(2,3)は必ず存在する地点
	t.field[2][2]=true;
	t.field[2][3]=true;

	//指定された種類(色)に応じて振り分け
	switch(k){
		case RED://赤
			t.field[3][1]=true;
			t.field[3][2]=true;
			break;
		case GREEN://緑
			t.field[1][1]=true;
			t.field[1][2]=true;
			break;
		case PURPLE://紫(T字)
			t.field[1][2]=true;
			t.field[3][2]=true;
			break;
		case BLUE://青(逆L字)
			t.field[2][1]=true;
			t.field[1][3]=true;
			break;
		case ORANGE://橙(L字)
			t.field[2][1]=true;
			t.field[3][3]=true;
			break;
		case YELLOW://黄(■)
			t.field[1][2]=true;
			t.field[1][3]=true;
			break;
		case SKY://空(｜)
			t.field[2][0]=true;
			t.field[2][1]=true;
			break;
		default://不明->エラー
			throw "int CreateTetriMino()::k error";
			break;
	}

	//FieldToBlk
	//フィールド情報から各ブロックの座標を設定する
	//tそのもののベース点はフィールドの(2,2)の座標とする
	int i=0;
	for(int x=0;x<5;x++){
		for(int y=0;y<5;y++){
			if(t.field[x][y]){
				t.blk[i]=Block(Point(t.x+BS*(x-2),t.y+BS*(y-2)),k);
				i++;
			}
		}
	}

	//設定したtを返す
	return t;
}
fieldは5*5の座標平面のように考えてください。


中心付近の(2,2),(2,3)はどのテトリミノでも使います。
それ以外の部分を各色によって振り分けています(図はRED)

そうしてfieldを設定し終わったら、そのfieldを元に4つのblkの座標を設定します。
t自体の座標はfieldの(2,2)の位置にあるとして(緑○)そこを基準にします。

それ以外はコンストラクタで0フィルしてるので大丈夫です。


//FieldToBlkとコメントしてあるのは何故かというと、

//func.cpp
#include "../include/main.h"

//フィールド情報から各ブロックの座標を設定する
//tそのもののベース点はフィールドの(2,2)の座標とする
bool FieldToBlk(TetriMino& t){
	int i=0;
	for(int x=0;x<5;x++){
		for(int y=0;y<5;y++){
			if(t.field[x][y]){
				t.blk[i].x=t.x+BS*(x-2);
				t.blk[i].y=t.y+BS*(y-2);
				i++;
			}
		}
	}
	return (i==4);
}
